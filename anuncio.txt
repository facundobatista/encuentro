
Encuentro 0.9

Encuentro es un simple programa que permite buscar, descargar y ver contenido del Canal Encuentro, Paka Paka, BACUA, Educ.ar y otros.

Notar que este programa no distribuye contenido directamente, sino que permite un mejor uso personal de esos contenidos.  Por favor, referirse a los sitios web correspondientes para saber qué se puede y qué no se puede hacer con los contenidos de tales sitios.

http://encuentro.taniquetil.com.ar/imgs/title.png

Página oficial:

    http://encuentro.taniquetil.com.ar/


La versión 0.9 trae los siguientes cambios con respecto a la versión anterior:

- Hay un nuevo backend! Banco Audiovisual de Contenidos Universales Argentino http://www.bacua.gob.ar/

- Se muestran imágenes para cada episodio

- Hay una cola explícita de descargas, mostrando los estados de todas las transferencias

- Las descargas terminadas se pueden filtrar por separado

- Mejor manejo de las cancelaciones, más logging, y otras pequeñas mejoras y correcciones

La forma más fácil de instalarlo, si tienen un Debian o Ubuntu, es usando este instalador:

    http://launchpad.net/encuentro/trunk/0.9/+download/encuentro-0.9.deb


Instalarlo en Arch es sencillo ya que está disponible en AUR, por ejemplo se puede hacer:

    yaourt -S encuentro


También se puede instalar directamente desde PyPI:

    sudo easy_install encuentro


En cualquier otro caso, pueden usar el tarball para instalarlo:

    http://launchpad.net/encuentro/trunk/0.9/+download/encuentro-0.9.tar.gz


Si tienen Windows, Fedora o Red Hat, todavía no tenemos un instalador, pero lo pueden instalar o usar desde el tarball, como acabo de indicar.

Si quieren obtener automaticamente las nuevas actualizaciones, pueden suscribirse al PPA; en una linea de comandos hagan:

    sudo add-apt-repository ppa:facundo/encuentro
    sudo apt-get update
    sudo apt-get install encuentro


Recuerden revisar el archivo AYUDA.txt si tienen alguna duda de cómo usar el programa.
